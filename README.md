# SAECTA Web Dashboard 🎓

Painel administrativo desenvolvido para a gestão do sistema educacional, controle de entidades (escolas, universidades) e acompanhamento do sistema de correção de avaliações.

## 🚀 Tecnologias Utilizadas

- **React.js** com **TypeScript**
- **Vite** (Build e Desenvolvimento rápido)
- **Ant Design (antd)** (Componentes de Interface)
- **React Router** (Navegação)

## 📦 Como rodar o projeto localmente

1. Clone este repositório
2. Instale as dependências executando:
   \`npm install\`
3. Inicie o servidor de desenvolvimento:
   \`npm run dev\`

## 🛠️ Estrutura e Padrões
- Utilização de `masonry-grid` para os cards de entidades.
- Interface responsiva com suporte a Dark Mode.
- Roteamento com passagem de estado (State) para reaproveitamento de telas (Cadastro/Edição).
